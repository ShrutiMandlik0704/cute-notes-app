# 🌸 Cute Notes App

A simple and cute **Notes Web Application** built using **Python Flask**.
Users can add, edit, and delete notes with a cute pink UI.

---

## ✨ Features

* ➕ Add new notes
* ✏️ Edit notes
* 🗑 Delete notes
* 💗 Cute girly interface
* 💾 SQLite database storage

---

## 🛠 Tech Stack

* Python
* Flask
* SQLite
* HTML
* CSS

---

## 🚀 How to Run

1. Install Flask

```
pip install flask
```

2. Run the application

```
python app.py
```

3. Open in browser

```
http://127.0.0.1:5000
```
